# RDT-Depo-Takip
RDT Pro, modern işletmelerin depo ve envanter yönetim süreçlerini dijitalleştirmek, hızlandırmak ve hata payını minimize etmek için geliştirilmiş, yüksek performanslı bir masaüstü uygulamasıdır. Python tabanlı olup, kullanıcı dostu CustomTkinter arayüzü ve güvenilir SQLite veritabanı altyapısını kullanır.
